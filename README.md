# Matrix Calculator

Простий калькулятор матриць на Python.

## Функціональність
- Додавання матриць
- Множення (на скаляр та матриці)
- Транспонування
- Юніт-тести
- Автоматична документація

## Використання
```python
from matrix import Matrix

A = Matrix([[1, 2], [3, 4]])
B = Matrix([[5, 6], [7, 8]])

print(A + B)  # Додавання
print(A * 2)  # Множення на скаляр
print(A.transpose())  # Транспонування
